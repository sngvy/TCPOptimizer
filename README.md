# TCPOptimizer

**TCPOptimizer — это скрипт, который автоматически применяет твик txqueuelen и устанавливает оптимальный конфиг sysctl для снижения задержек при туннелировании трафика.**

# Использование

**`bash <(curl -Ls https://raw.githubusercontent.com/sngvy/TCPOptimizer/refs/heads/main/TCPOptimizer.sh)`**
